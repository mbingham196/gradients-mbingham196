"To create a linear gradient you must define at least two color stops. Color stops are the colors you want to render smooth transitions among. You can also set a starting point and a direction (or an angle) along with the gradient effect."

This is an example of a default linear gradient. "linear-gradient(red, yellow)"

Colors are not the only parameter we can use as a stop. We can also use opacity by using the rgba formula. (red, green, blue, alpha) Where Alpha signifies opacity. linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1)

This is already cool, but you can also use more than one color by listing them in the order you want them to appear. "linear-gradient(90deg, red, orange, yellow, green, blue)"

The colors also don't have to appear in even swaths. You can use percentages to define where you want the breaks to be. "linear-gradient(90deg, red 10%, orange 20%, yellow 30%, green 40%, blue 50%)"

The next type of gradient is a Radial Gradient which is where you start with one color and then radiate that color out from a point. "radial-gradient(green, blue)"

You can also specify the point of origin for the gradient. by specifying the pixels for the X and Y axis. radial-gradient(farthest-corner at 40px 40px, green 0%, blue 100%);
